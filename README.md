# README

Visual Studio Code 上で rubocop を実行します。

execute rubocop on Visual Studio Code

# Configuration

Speciry to config ruby.rubocop.executePath

```
{
	"ruby.rubocop.executePath": "D:/bin/Ruby22-x64/bin/",
	"ruby.rubocop.onSave": true // default true
}
```
